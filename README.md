# MineClearing
Android 扫雷小游戏- compose版

### 设计思路

1. 在屏幕上绘制出指定大小的N x N方格 done
2. 在部分方格下“埋雷” done
3. 在每个“地雷”附近的方格内增加提示信息 done
4. 将所有方格隐藏 done
5. 用户点击之后揭晓是否为“地雷” 
6. 游戏结束：所有的非地雷被打开或者用户“踩雷”
